Usage Guide

  1. Sign in with Google

  2. Use the File Upload Interface on the left to upload PDF documents (max 50 MB) you would like to feed to Queread

  3. Press the Upload button to feed the selected PDF document to Queread

  4. Ask Queread questions related to the PDF documents that were uploaded

  5. You can also delete the uploaded PDF documents to make Queread forget about them
